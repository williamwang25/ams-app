# 05 管理端功能

> **作用**：拆分管理端 Web 各模块的页面、交互与验收点。
> **读者**：管理端 agent（`admin/src/modules/*` 负责人）。
> **编辑边界**：可调整页面细节、组件选型；模块边界与已锁决策不可改。

## 5.1 模块总览

| 模块 | 路由 | 说明 |
|------|------|------|
| 登录 | `/login` | 账号密码登录 |
| 看板 | `/` | Dashboard |
| 资产 | `/assets` `/assets/new` `/assets/:id` | 列表 / 入库 / 详情 |
| 借用审批 | `/borrows` `/borrows/:id` | 待审批与历史 |
| 大型资产 | `/large-assets` | 大型资产专题页 |
| 通知公告 | `/notices` | 管理通知 |
| 用户管理（超管） | `/admins` | 教师账号管理 |
| 报表（预留） | `/reports` | 空页占位 |
| 闲置共享（预留） | `/share` | 空页占位 |

## 5.2 登录

- 字段：用户名、密码、"记住我"。
- 调云函数 `auth.adminLogin`，成功后存 token 到 localStorage，路由跳转 `/`。
- 路由守卫：未登录访问受保护路由跳 `/login`。
- 错误提示：账号 / 密码错误、账号禁用、网络错误。

**验收点**：

- 登录态过期或被踢出后自动跳回登录。
- 密码不在前端日志 / 网络面板中明文暴露（云函数内部 hash）。

## 5.3 Dashboard（首页看板，已锁）

### 5.3.1 顶部指标卡（5 张）

| 卡片 | 数据源 |
|------|--------|
| 资产总数 | `dashboard.summary` |
| 资产总值 | 同上 |
| 使用中 | `business_status=IN_USE` 计数 |
| 出借中 | `business_status=LENT` 计数 |
| 待审批 | `ams_borrow_request.status=PENDING` 计数 |

### 5.3.2 通知公告区

- 显示最近 N 条已发布通知（`ams_notice.published=true`）。
- 点击进入通知详情或管理页。

### 5.3.3 总账信息

- 按学院 / 部门聚合：固定资产数量、总金额、当前借出数量。
- 表格 + 简单筛选（部门下拉）。

### 5.3.4 出入仓统计

| 维度 | 指标 |
|------|------|
| 数量 | 出仓（借出）总量 / 本月 / 今日 |
| 金额 | 出仓金额总值 / 本月 / 今日 |

> "出仓"= 借出事件次数 / 金额累计；"入仓"= 归还事件。文档命名以"出入仓"为统一术语。

### 5.3.5 图表（4 张）

| 图 | 类型 | 说明 |
|----|------|------|
| 按部门资产分布 | 柱状 | x=部门，y=资产数量（可切换为金额） |
| 按状态占比 | 饼状 | 业务状态枚举占比 |
| 出入仓数量曲线 | 折线 | 近 30 / 90 天每日借出 + 归还 |
| 出入仓金额曲线 | 折线 | 同上，y 为金额 |

> 图表库由 agent 选型（ECharts / Chart.js / Recharts 等），但要支持响应式、保持主色 `#0096C2`。

**验收点**：

- 所有指标点击后可跳转到对应明细列表（如点"待审批"跳借用审批列表）。
- 时间窗口可切换（7/30/90 天）。
- 接口慢加载时显示骨架屏，不阻塞页面。

## 5.4 资产管理

### 5.4.1 列表 `/assets`

- 筛选：关键字（名称 / 编号 / 品牌 / 型号）、部门、业务状态、是否大型、入库时间区间。
- 列：图片缩略图、`asset_no`、`name`、`brand`、`spec`、`dept_name`、`user_name`、`location_name`、`business_status` 标签、`unit_price`、操作（详情 / 编辑 / 变动 / 报废）。
- 分页 20 条 / 页；支持导出当前筛选（CSV，可选）。

### 5.4.2 入库 `/assets/new`

- 单条表单，分组展示（基础 / 财务 / 归属 / 取得 / 业务）。
- 图片上传（多张，云存储）。
- 资产编号：默认置灰提示"系统将根据国标分类自动生成"，可由超管手动覆盖。
- 提交调用 `asset.create`。

### 5.4.3 详情 `/assets/:id`

- 顶部：图片轮播、基础信息、`business_status` 标签、当前借用 / 使用人。
- 标签页：
  - **详细字段**：全字段只读展示。
  - **生命周期 Timeline**：来自 `ams_asset_log`，按时间倒序，展示操作类型、变更字段、操作人、时间。
  - **借用记录**：来自 `ams_borrow_request`。
- 操作：编辑、变动位置、变动使用人、变更状态、报废。

**验收点**：

- 同一资产在不同业务状态下展示对应操作按钮（如 `LENT` 时不能改使用人）。
- Timeline 至少展示 `CREATE` / `UPDATE` / `BORROW` / `RETURN` / `STATUS_CHANGE` 等事件。

## 5.5 借用审批 `/borrows`

- Tab：待审批 / 已通过 / 已拒绝 / 已归还。
- 列：流水号、教师姓名、申请时间、资产数量、用途、拟归还日期、状态。
- 点击行进入详情：
  - 教师信息、签名图片预览、资产明细列表。
  - 操作按钮：通过 / 拒绝（拒绝需填原因）。
- 通过 / 拒绝调用 `borrow.approve` / `borrow.reject`。

**验收点**：

- 通过后资产 `business_status` 自动变 `LENT`；拒绝后回 `IDLE`。
- 拒绝原因写入申请单。

## 5.6 大型资产 `/large-assets`

- 筛选条件：`is_large=true`（阈值在字典中可调）。
- 指标卡：大型资产总数、总值、当前借出数、累计借出次数。
- 列表 + 详情同 5.4，但详情额外展示：
  - **使用率**：定义为 `已借出天数 / 入库后天数`，或近 N 个月借出次数。具体口径由 agent 选定，但需在页面注释中说明。
  - **借出 Timeline**：横向时间轴，按月份展示借出区间。

## 5.7 通知公告 `/notices`

- 列表 + 富文本 / Markdown 编辑器新建 / 编辑。
- 字段：标题、内容、级别（普通 / 重要）、发布开关。
- 发布即对教师端可见。

## 5.8 用户管理 `/admins`（超管）

一期优先管理教师账号（`ams_teacher`），管理员账号仍沿用 `auth.adminLogin` 固定超管。

- 教师列表：关键字（账号 / 姓名 / 手机 / 部门）、部门、微信绑定状态筛选。
- 新增教师：账号、初始密码、姓名、手机号、部门。
- 编辑教师：账号、姓名、手机号、部门。
- 重置密码：生成临时密码并展示给管理员。
- 解绑微信：清空教师 `openid` / `unionid` / `bound_at`，教师下次可重新账密绑定。
- 删除教师：若已有借用申请记录，不允许删除，避免历史单据失去归属。

**验收点**：

- 列表不展示 `password` 字段。
- 教师账号唯一，重复创建 / 改名需被拦截。
- 删除有借用历史的教师返回明确错误。
- 解绑后教师端 `teacherLoginByOpenid` 应返回未绑定，账密登录可重新绑定。

## 5.9 预留页面

- `/reports`：放置一个占位区，注明"AI 报表（二期）"。
- `/share`：放置一个占位区，注明"闲置 / 算力共享（二期）"。

## 5.10 通用 UX 规范

- 表格固定表头、可调列宽（可选）。
- 操作确认：报废 / 删除等不可逆操作需二次确认弹窗。
- Loading / Empty / Error 三态完整。
- 所有金额按千分位 + 两位小数展示。
